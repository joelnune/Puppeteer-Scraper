## Puppeteer Scraper

### Descrição:
Esse script tem como finalidade pegar todos os dados requisitados do website CRECI e salvar em um arquivo .csv

### Dependencias:
 "csv-writer": "^1.6.0",
 
 "puppeteer": "^22.12.1"

### Como Executar:
`` npm install ``

``` node script-florianopolis.js ```

### Output:
Arquivo ```data.csv``` com todos os dados do website.
